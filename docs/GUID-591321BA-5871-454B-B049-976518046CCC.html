3 MAC Layer Features

MAC Layer is responsible for following functional requirement of a standalone IEEE 802.15.4.
  • Transmission of data
  • Purge data
  • Reception of data
  • Control of PHY receiver
  • Timestamp of incoming data
  • Beacon management
  • Transmit beacons
  • Receive beacons
  • Channel access mechanism
  • Frame validation
  • Acknowledged frame delivery
  • Association and disassociation
  • Promiscuous mode
  • Security
  • Unsecured mode
  • Secured mode
  • Data encryption
  • Frame integrity
  • ED Scanning
  • Active scanning
  • Passive scanning
  • Orphan scanning
  • Store at least one transaction in indirect message queue
